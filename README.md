## Rule
1. 흑이 선공으로 돌 하나를 둠
2. 다음 턴부터는 돌을 서로 2개씩 둠
3. 먼저 6목을 만드는 사람이 win

## Install pygame for Mac
```
python3 -m venv myvenv
source myvenv/bin/activate
python -m pip install --upgrade pip
pip install pygame
```
